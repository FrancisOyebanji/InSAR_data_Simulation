README

Overview



Simulate deformation fields with spatial noise, instantaneous, and temporal deformation. Produces noisy datasets and plots.



**Files \& Functions**



**main\_simulation.py**

Set parameters, load observations, and run run\_full\_simulation.



**full\_simulation.py**

Full workflow:



Spatial noise (*simulate\_spatial\_noise.py*)



Instantaneous deformation (*simulate\_instantaneous\_deformation.py*)



Temporal deformation (*simulate\_temporal\_deformation.py*)



Add noise and visualize (*plot\_fields.py*)



**Usage**



Set **parameters** in main\_simulation.py



Run **main\_simulation.py** to generate fields and plots
